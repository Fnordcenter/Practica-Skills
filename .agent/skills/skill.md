# My Skill

Use this when you need to a new skill or are you asked for one.

This is helpful for having the ability to make new tasks and how to do them

Make sure the skills you make are generic and agnostic to any programming language it uses.



## When to use this skill

Use it whenever a user asks to "create a new skill" or "define a new skill."

## How to use it

1. Fill out the json frontmatter:

    name: Provide a concise, lower-case, hyphenated identifier (e.g., code-reviewer, story-outliner).

    description: Write a 1-2 sentence summary explaining exactly what the skill does and the specific scenarios that should trigger it.

2. Define the "When to use" section:

    List the specific conditions, user requests, or workflow states that indicate this skill should be activated.

3. Define the "How to use" section:

    Provide step-by-step instructions, conventions, and patterns the agent must follow when executing the skill.
